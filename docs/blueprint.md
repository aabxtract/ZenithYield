# **App Name**: YieldZenith

## Core Features:

- Wallet Connection: Enable users to connect their wallets using RainbowKit or Web3Modal.
- LP Token Staking: Allow users to stake LP (ERC20) tokens into the smart contract.
- Reward Accumulation: Calculate and display real-time rewards based on a fixed APY.
- Reward Claiming: Allow users to claim accumulated reward tokens at any time.
- LP Token Unstaking: Enable users to unstake their LP tokens.
- Dashboard Display: Show userâ€™s staked amount, accumulated rewards, and total value locked (TVL).
- APY Suggestion Tool: The LLM will use a tool that uses real time price of the LP token components, TVL and trading volume on different DEXs to find an optimal APY.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and stability.
- Background color: Light gray (#F5F5F5) for a clean, minimal look.
- Accent color: Bright cyan (#00BCD4) to highlight key actions and information.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral feel.
- Use simple, outlined icons for clarity and a DeFi aesthetic.
- Maintain a clean, well-spaced layout to enhance readability.
- Use smooth transitions and subtle animations to provide feedback.
